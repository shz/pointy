Pointy Documentation

Alas, entirely unwritten. The actual operation is fairly straightforward, however; take a look at the examples (they're painless, I promise) to get an idea of how to use it. From there you can pretty much just rely on XML docs to get the hang of the rest.

If you're building an application that makes use of some more "serious" HTTP features you'll more than likely end up making use of Pointy.Util, which provides the following classes:

  • FormUrlencoded — Decodes/encodes data using x-www-form-urlencoded
  • MimeType — Provides MIME header parsing/generation tools
  • UrlEncoding — Decodes/encodes data using URL encoding
